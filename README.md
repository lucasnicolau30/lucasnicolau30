### 🇺🇸 About me

I'm a 5th-semester Software Engineering student at UFAM (Federal University of Amazonas), with hands-on experience as a **Full Stack Developer** and **QA/Test Automation Engineer**.

At **RedMaxx**, I built a custom test automation framework in Python + Pytest for REST APIs, achieving **98% route accuracy** across positive/negative scenarios, status codes, and JSON structure validation. I also implemented automatic CSV report generation for traceability.

I volunteered as a Full Stack Developer on **PACE 2025** (a university extension project), building a real-world web application integrated with the Google Sheets API — used by UFAM's School of Dentistry in scientific research.

I'm also a former **Teaching Assistant** for Algorithms & Data Structures I and Linear Algebra I at UFAM.

I'm fluent in English, focused on backend, clean code, and building things that actually work in production.
